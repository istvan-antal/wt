Wt

an all in one web developer helper tool

Download Download fork Wt on GitHub

Description

WebTool is basically a collection of tools that I use for mostly client side web development, basically it includes:

  • JSLint
  • JSHint
  • Closure compiler
  • JSDoc
  • and some other goodies
Available commands:

`wt module create namespace <modulename>`
`wt module create object <modulename>`
`wt module create function <modulename>`
`wt module create widget <modulename>`
`wt lint <file1.js file2.js ...>`
`wt hint <file1.js file2.js ...>`
`wt doc create <file1.js file2.js ...> <documentation directory>`
`wt compile <file1.js file2.js ...>  <output file>`
`wt build <file1.js file2.js ...>  <output file>`

Experimental:
`wt acompile <file1.js file2.js ...>  <output file>`

`wt help` - for more details...

References:
JSDOC - http://code.google.com/p/jsdoc-toolkit/
Google Closure Compiler - http://code.google.com/closure/compiler/
JSLint - http://www.jslint.com/

Installation

WT is written in PHP and requires Java for some of the tools that it includes.

Ubuntu

$ wget https://github.com/downloads/istvan-antal/wt/wt-current-all.deb
$ sudo dpkg -i wt-current-all.deb
$ rm -f wt-current-all.deb
